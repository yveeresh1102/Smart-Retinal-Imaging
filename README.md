# Smart Retinal Imaging: AI-based Vascular Health Assessment

This project uses deep learning and machine learning to analyze retinal images and assess cardiovascular risk.

## Features

- CLAHE image enhancement
- ResNet34-UNet vessel segmentation
- Hybrid vascular feature extraction
- Cardiovascular risk classification
- Streamlit web application

## Technologies

- Python
- PyTorch
- OpenCV
- Streamlit
- NumPy
- LightGBM

## Workflow

Retinal Image → Preprocessing → Vessel Segmentation → Feature Extraction → Risk Prediction

## Results

Dice Score: 0.84  
Pixel Accuracy: 91.78%  
Classification Accuracy: 83%
